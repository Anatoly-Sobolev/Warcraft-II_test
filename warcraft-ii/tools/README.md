# tools

> ↑ [Корень проекта](../README.md) · 🏛 [Архитектура](../docs/architecture/architecture.md)

**Ответственность:** инструменты разработки и пайплайна контента (запускаются вне игры).

## Файлы
| Файл | Назначение |
|---|---|
| `content_validation_tool.gd` | Проверка ID, ссылок, требований, локализации, порядка кампаний. |
| `map_baker.gd` | Авторская карта → компактные логические массивы. |
| `atlas_validation_tool.gd` | Проверка регионов, размеров, ключей анимаций. |

## Подпапки

| Папка | Назначение |
| --- | --- |
| [`import/`](import/README.md) | Будущие инструменты чтения локальных Warcraft II/Wargus reference-only источников и генерации наших reports/черновиков. |
