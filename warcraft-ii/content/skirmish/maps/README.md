# Карты схватки — соглашение об именах

Каждая карта схватки — это пара файлов:

```
<map_id>_logic.tres    # логическая карта для Warcraft Runtime
<map_id>_visual.tres   # визуальная карта для Presentation
```

`skirmish_manifest.tres` (на уровень выше) хранит лёгкий список доступных карт:
`map_id`, имя, размер, число слотов. Тяжёлые ресурсы карты подгружаются только
когда игрок выбрал её в настройке матча — каталог не тянет за собой все карты сразу.

`example_map` — карта-образец. Новые карты добавляются по тому же шаблону.
