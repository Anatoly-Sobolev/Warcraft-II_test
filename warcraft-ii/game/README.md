# game

> ↑ [Корень проекта](../README.md) · 🏛 [Архитектура](../../ARCHITECTURE.md)

**Ответственность:** всё, что живёт внутри игрового сеанса (матча). Это не один
модуль, а слой из шести модулей. Состоянием владеют дочерние модули, не `game/`.

## Подмодули
| Модуль | Ответственность | README |
|---|---|---|
| `campaign` | Прогресс между миссиями, открытие миссий | [→](campaign/README.md) |
| `match` | Сборка и жизненный цикл одного матча (composition root) | [→](match/README.md) |
| `input` | Жесты, выбор, превращение действий в команды | [→](input/README.md) |
| `simulation` | **Авторитетный игровой мир** (логика, данные) | [→](simulation/README.md) |
| `scenario` | Миссии, сюжет/кат-сцены, обучение | [→](scenario/README.md) |
| `presentation` | Спрайты, камера, туман, миникарта, игровой звук | [→](presentation/README.md) |

## Поток внутри матча
```
input ──команды──┐
scenario ────────┼──► simulation ──┬── события ──► scenario
ai (в simulation)┘                 ├── dirty ────► presentation
                                   └── ViewData ─► ui (вне game/)
```

`match` создаёт эти модули и соединяет их типизированными портами. Модули не ищут
друг друга через дерево сцен.
