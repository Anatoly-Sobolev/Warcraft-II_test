# simulation/navigation

> ↑ [simulation](../README.md) · 🏛 [Архитектура](../../../docs/architecture/architecture.md)

**Ответственность:** поиск пути по сетке (A*) с очередью запросов, бюджетом на такт
и кэшем, учитывающим версии регионов карты.
**Владелец состояния:** очередь запросов и кэш путей (производные данные — в снимок не сохраняются, перестраиваются).

## Файлы
| Файл | Назначение |
|---|---|
| `navigation_service.gd` | Приём запросов, применение бюджета такта. |
| `grid_pathfinder.gd` | A* для земли/воды/воздуха. |
| `path_request_queue.gd` | Распределение расчётов между тактами. |
| `path_cache.gd` | Кэш: слой движения + версии регионов пути. |
| `group_path_planner.gd` | Разные конечные клетки и общие участки для групп. |
| `stuck_resolver.gd` | Смена локальной цели/пересчёт/завершение невозможного приказа. |

## Инварианты
- Кэшированный путь годен, только если текущие `PathRegionVersions` совпадают с записанными.
- Полный обход кэша при изменении карты не нужен — устаревшая запись просто отбрасывается.
- Зависит от `../spatial/` (сетки и версии регионов).
